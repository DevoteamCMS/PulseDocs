# Cloud Inventory

## Overview

A complimentary multi-cloud inventory tool requiring no credit card or hidden fees. The platform offers rapid visibility across cloud environments through several key sections:

- **Dashboard** — Summary page with clickable navigation to other areas
- **Potential Savings by Devoteam CSP** — Identifies pay-as-you-go Azure subscriptions and proposes cost reduction strategies
- **Costs** — "Multi-Cloud Spend Visibility" providing transparency into usage and billing across subscriptions and services
- **Recommendations** — "All Recommendations in one page" displaying unified multi-cloud suggestions
- **Security** — "Security Alerts" consolidating critical cloud alerts for monitoring
- **Assets** — "Assess Your Infrastructure" revealing operational overhead and SaaS service complexity
- **Locations** — "Audit Data Compliance" verifying architectural adherence and data governance
- **Tags** — "Tag Governance Overview" evaluating asset management and cost allocation strategies

## Cloud Costs

The cost section emphasizes period summaries, monthly totals, and savings opportunities. Users can aggregate expenses across 6 or 12-month periods and organize by cloud provider, tenant, subscription, resource group, or service type.

**Cost navigation tips:**
- Search by subscription or resource group naming for business unit allocation
- Export tables as CSV files
- Toggle "Hide other" to display only visible table items

## Recommendations

This section synthesizes recommendations from "Azure Advisor, AWS Trusted Advisor, Google Recommenders" into a unified view with impact summaries and grouped lists. Users can identify affected resources and export findings.

## Security Alerts

Aggregates alerts from Azure Defender for Cloud, AWS GuardDuty, and Google Security (threat-type findings) with resource tracking and CSV export capabilities.

## Assets

Tracks resource changes over time and categorizes services by type, noting when IaaS-heavy environments warrant modernization toward PaaS or SaaS solutions.

## Locations

Displays regional deployment distribution across cloud provider data centers to validate compliance and architectural consistency.

## Tags

Assesses tagging usage patterns and identifies the top five tags to highlight governance gaps.
