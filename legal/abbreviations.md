# Abbreviations

PULSE - current product name, the same as CMP

INSIGHT - old product name, the same as CMP

CMP - Cloud Management Platform a Devoteam SaaS type of application with frontend for customers

GCP - Google Cloud Provider

SA - Service Account, naming used by Google Cloud

MS - Microsoft

Azure - Microsoft Azure Cloud

SPN - Service Principal Name, used by Azure, it is also called Application Registration, also known as Service Account

CSP - Cloud Service Provider - Microsoft Azure Cloud billing type

EA - Enterprise Agreement - Microsoft Azure Cloud billing type

AWS - Amazon Web Services

IAM user - AWS IAM user created for programmatic access, also known as Service Account

API - An application programming interface

URL - Uniform Resource Locator, is the address of a unique resource on the internet. Internet address that is visible in browser address bar (field)
